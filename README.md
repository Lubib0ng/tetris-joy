## 프로젝트 소개
HTML5 Canvas를 활용하여 제작한 테트리스 게임입니다.  
플레이어는 떨어지는 테트로미노 블록을 조작하여 가로 줄을 완성하고,  
라인을 제거하며 점수를 획득하는 것을 목표로 합니다.

## 개발 기간
2025.12.19(개인 프로젝트)

##주요 기능
- 테트리스 기본 블록 (i,o,t,s,z,j,L) 구현
- 블록 이동 / 회전 / 하드 드롭
- 라인 완성시 자동 제거
- 점수및 게임 오버 처리
- 실시간 화면 렌더링
- 콤보 및 최고 점수 표시
- 웹 브라우져에서 바로 실행 가능

##사용 기술
- Frontend: HTML, CSS, JavaScript
- Rendering: HTML5 Canvas
